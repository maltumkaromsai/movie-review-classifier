# Movie Review Sentiment Classifier

A machine learning project for classifying movie reviews as **positive** or **negative** using **Support Vector Machine (SVM)**.

## Project Objective
The objective of this project is to build a sentiment analysis model that can automatically classify movie reviews based on their textual content.

## Tools & Libraries Used
- Python
- Pandas
- NumPy
- Scikit-learn
- NLP (Text Preprocessing)

## Workflow
- Data loading
- Text preprocessing
- Feature extraction
- Model building
- Training multiple SVM models
- Model comparison
- Evaluation

## Models Compared
- Linear SVM
- Polynomial Kernel SVM
- RBF Kernel SVM

## Outcome
Different SVM kernels were trained and compared to evaluate their performance on movie review sentiment classification.

## Project File
- `movie_review_classifier.ipynb` → Main notebook containing preprocessing, training, model comparison, and evaluation.
