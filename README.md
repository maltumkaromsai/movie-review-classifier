# Movie Review Sentiment Classifier

A machine learning project for classifying movie reviews as **positive** or **negative** using **Support Vector Machine (SVM)**.

## Project Objective
The objective of this project is to build a sentiment analysis model that can automatically classify movie reviews based on their textual content.

## Tools & Libraries Used
- Python
- Pandas
- NumPy
- Scikit-learn
- NLP (Text Preprocessing)

## Workflow
- Data loading
- Text preprocessing
- Feature extraction
- Model building
- Model training
- Evaluation

## Model Used
- Support Vector Machine (SVM)

## Outcome
The model was trained to classify movie reviews into positive and negative sentiment categories.

## Project File
- `movie_review_classifier.ipynb` → Main notebook containing preprocessing, training, and evaluation steps.
